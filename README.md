# 🌤 React Weather App

A responsive Weather Forecast Application built using ReactJS and OpenWeather API. This application allows users to search for any city and view real-time weather information including temperature, humidity, wind speed, and weather conditions.

## 🚀 Features

* Search weather by city name
* Real-time weather data using OpenWeather API
* Display temperature in Celsius
* Display humidity percentage
* Display wind speed
* Display weather conditions
* Responsive and user-friendly interface
* Error handling for invalid city names
* Clean and modern UI design

## 🛠 Technologies Used

* ReactJS
* JavaScript (ES6+)
* CSS3
* OpenWeather API
* Vite
* Git & GitHub

## 📚 React Concepts Implemented

This project demonstrates the following ReactJS concepts:

* Functional Components
* React Hooks (`useState`)
* Event Handling
* API Integration using Fetch API
* Conditional Rendering
* State Management
* Form Handling
* Dynamic UI Updates

## 💡 Project Workflow

1. User enters a city name.
2. React captures the input using state management.
3. Fetch API sends a request to OpenWeather API.
4. Weather data is retrieved and processed.
5. Results are displayed dynamically on the screen.
6. Appropriate error messages are shown for invalid inputs.
